Serviço de Autenticação

O Serviço de autenticação permite fazer o registo de utilizadores para uma base de dados em mysql através de uma comunicação REST API e fazer a respectiva validação de autenticação (login).

Serviço de Reserva

O serviço de reserva proporciona por intermédio de uma base de dados, por comunicação REST API e atualização de um booleano fazer as reservas.

Serviço de Horário

O serviço de horário permite adicionar datas e horas a uma base de dados. Cada sala pode ter vários horas no mesmo dia e vários dias com a mesma hora.

Serviço de Pedidos

O serviço concede ao utilizador enviar uma mensagem para efectuar pedidos ou reclamações. A mensagem é enviada por intermédio do rabbitMQ e guardada num ficheiro de texto.
